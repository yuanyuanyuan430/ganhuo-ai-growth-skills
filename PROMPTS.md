# Prompts for 干活 AI Growth Skills

这些 prompt 可以直接复制到 Codex / 干活 AI 中使用。每个 prompt 都会把一个增长问题变成可执行 brief、检查表或路线图。

## 1. AI 搜索可见度 Baseline

```text
使用 $ganhuo-llm-citation-growth，帮我做一个 AI 搜索可见度 baseline。

品牌/产品：
官网：
目标市场：
希望被推荐的场景：
主要竞品：
已有内容资产：

请输出：
1. 20 个 ChatGPT / Gemini / Perplexity 可复测 prompt；
2. 当前 mention / citation / sentiment / accuracy 记录表；
3. 竞品被引用的信源地图；
4. 我们最该补的 owned content 和 off-site signal；
5. 30 天行动计划和复测节奏。
```
