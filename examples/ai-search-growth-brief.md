# AI Search Growth Brief 示例

> 示例场景：一个面向海外创作者的 AI 写作工具，希望在 ChatGPT、Gemini、Perplexity 等 AI 答案里更容易被发现、理解、推荐和引用。

## Target Recommendation Moment

| Moment | User question | Desired association |
| --- | --- | --- |
| Tool selection | What AI writing tool should I use for multilingual blog drafts? | Brand appears as a practical option for multilingual creators |
| Workflow design | How can a small team publish AI-assisted articles faster? | Brand is associated with repeatable editorial workflows |
| Trust check | Which AI writing tools support fact checking and human review? | Brand is connected to review, source notes, and editorial control |

## Baseline Prompts

```text
What are the best AI writing tools for small content teams?
Which AI writing tools are good for multilingual SEO content?
How should a creator build an AI-assisted publishing workflow?
What tools help turn research into blog drafts with human review?
Which AI writing products are recommended for content operations?
```

## Current Visibility Table

| Platform | Mentioned? | Cited? | Competitors appearing | Notes |
| --- | --- | --- | --- | --- |
| ChatGPT | No | No | Jasper, Copy.ai, Notion AI | Brand entity not clear enough |
| Gemini | Partial | No | Grammarly, Jasper | Needs stronger comparison and docs pages |
| Perplexity | No | No | HubSpot, Zapier, Jasper | Needs sourceable pages and third-party mentions |

## Source Gap Map

| Gap | Why it matters | Asset to build |
| --- | --- | --- |
| No category page | AI cannot infer primary use case | `/ai-writing-workflow/` |
| Weak comparison content | Competitors dominate recommendation context | `Brand vs Jasper vs Copy.ai` |
| No workflow examples | Hard to cite practical usage | 3 workflow case studies |
| No Reddit / forum discussion | Lacks public user-language signal | Community participation plan |
| Thin docs | Agent cannot extract features reliably | `llms.txt`, docs index, FAQ |

## 30-Day Action Plan

| Week | Action | Output |
| --- | --- | --- |
| Week 1 | Run GEO technical audit and create prompt baseline | Crawl/readability fixes, baseline sheet |
| Week 2 | Publish category page and workflow guide | Citation-ready owned pages |
| Week 3 | Add comparison page and FAQ | Better recommendation context |
| Week 4 | Start Reddit/community participation and retest prompts | Off-site language signals and updated visibility |

## Retest Schedule

- Test the same 20 prompts every Friday.
- Record mention, citation, sentiment, accuracy, and competitor movement.
- Update owned pages when AI answers misunderstand positioning.
- Add new prompts when customers use new language to describe the category.
