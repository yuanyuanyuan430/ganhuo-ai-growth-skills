# 干活 E-E-A-T 内容升级

## Overview

Use this skill to improve substance before polishing wording.

## Workflow

1. Identify the user's goal, target query, audience, and current draft weaknesses.
2. Check factual claims and mark unsupported, stale, generic, or overconfident statements.
3. Add first-hand evidence: test notes, screenshots, data, interviews, product usage, expert review, or real examples.
4. Strengthen trust signals: author context, methodology, sources, limitations, update date, and disclosure.
5. Improve usefulness: decision criteria, examples, table, FAQ, caveats, and next actions.
6. Create distribution variants only after the core article is stronger: social post, infographic outline, newsletter, or audio script.

## Output

Return an upgrade plan, revised outline, missing evidence list, and final edit priorities.

## References

Read `references/source-map.md` when you need source provenance, material-specific guardrails, or the original file list behind this skill.

## Boundaries

- Do not fabricate first-hand experience, credentials, reviews, or data.
- Do not treat E-E-A-T as keyword sprinkling.
