# 下海 AI 联盟站

## Overview

Use this skill to create a minimum viable affiliate site that is original, reviewable, and not just a copied bridge page.

## Workflow

1. Define niche, audience, offer type, traffic channel, monetization path, and review deadline.
2. Analyze reference sites for structure only; rewrite positioning, copy, visual direction, and proof points from scratch.
3. Require core trust pages: About, Contact, Privacy, Terms, Affiliate Disclosure, and clear editorial policy where relevant.
4. Build the first content set around useful buyer questions, comparison criteria, FAQs, and transparent affiliate context.
5. Verify mobile usability, page speed, crawlability, broken links, form behavior, and deploy URL.
6. Create a launch checklist for affiliate-network review and post-launch tracking.

## Output

Return site blueprint, page list, trust/compliance checklist, first content plan, and launch blockers.

## References

Read `references/source-map.md` when you need source provenance, material-specific guardrails, or the original file list behind this skill.

## Boundaries

- Do not clone another site's brand, visual identity, copy, or proprietary assets.
- Do not promise revenue; evaluate whether the site can plausibly pass review and serve users.
