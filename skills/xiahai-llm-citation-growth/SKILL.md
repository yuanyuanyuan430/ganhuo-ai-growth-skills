# 下海 LLM 引用增长

## Overview

Use this skill to move from 'AI should recommend us' to a measured source-influence plan.

## Workflow

1. Build a prompt library covering buyer questions, comparison prompts, alternatives, use cases, and dealbreakers.
2. Run or request baseline answers across target AI/search platforms; record mentions, citations, sentiment, accuracy, and competitor sources.
3. Create a truth notebook: what the brand is, what it is not, strongest proof, comparisons, limitations, and canonical sources.
4. Map cited source types: owned pages, Reddit, YouTube, reviews, listicles, docs, PR, forums, and knowledge bases.
5. Repair owned content at passage level: answer snippets, entity clarity, comparison blocks, sourceable data, and citations.
6. Plan off-site signals where appropriate: honest reviews, community answers, expert quotes, listicle inclusion, and PR assets.
7. Retest on a schedule and report visibility, citation rate, truth score, and referral changes.

## Output

Return baseline matrix, source gap map, content/off-site action plan, and retest schedule.
