# 下海 小语种 SEO

## Overview

Use this skill to avoid lazy translation pages and build locally credible SEO pages.

## Workflow

1. Select candidate markets by product fit, search demand, competition, logistics/payment feasibility, and content resources.
2. Research keywords directly in the target language; do not translate the English keyword list blindly.
3. Inspect local SERPs for page types, local competitors, marketplaces, forums, regulations, and trust signals.
4. Choose localization depth: native original, expert-translated, machine draft plus local expert review, or no-go.
5. Plan pages with local examples, units, currency, compliance claims, reviews, and buying context.
6. Set QA: back-translation, native reviewer notes, schema/hreflang, local internal links, and performance tracking.

## Output

Return market shortlist, keyword clusters, localization plan, page requirements, and launch risks.

## Boundaries

- Do not present machine translation as native expertise.
- Do not ignore local legal, certification, payment, or shipping context.
